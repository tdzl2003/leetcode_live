# 奇数个 a 和任意个其它字符组成的字符串

```mermaid
graph LR
    Start((Start))
    1((1))
    2((2))
    End((End))

    1 -- a --> 2
    1 -- "[^a]" --> 1
    2 -- a --> 1
    2 -- "[^a]" --> 2
    2 -- $ --> End
```

```mermaid
graph LR
    Start((Start))
    1((1))
    2((2))
    End((End))
    Start -- ^ --> 1
    1 -- "[^a]*a" --> 2
    2 -- "[^a]*a" --> 1
    2 -- "[^a]*$" --> End
```

```mermaid
graph LR
    Start((Start))
    1((1))
    2((2))
    End((End))

    1 -- "[^a]*a" --> 2
    1 -- "[^a]*a[^a]*a" --> 1

```

```mermaid
graph LR

    1((1))
    2((2))
    End((End))

    2 -- "[^a]*$" --> End
```

```mermaid
    Start((Start))
    End((End))

```
